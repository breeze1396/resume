## 🚀 功能特性

- 📱 响应式设计，支持各种设备
- 🌍 多语言支持（中文/英文）
- 🎨 现代化UI设计
- 📄 PDF简历下载

## 🛠️ 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 国际化 (i18n)

## 📦 本地运行

1. 克隆项目
```bash
git clone https://github.com/breeze1396/resume.git
cd resume
```

2. 使用本地服务器运行
```bash
# Python 3
python -m http.server 8000

# 或者使用 Node.js
npx serve

# 或者使用 PHP
php -S localhost:8000

# 右键打开 index.html 文件也可以直接预览
```

3. 在浏览器中访问 `http://localhost:8000`

## 📝 自定义

1. 修改 `locales/translations.js` 更新内容
2. 修改 `css/` 目录下的样式文件自定义样式
3. 替换 `images/` 目录下的图片
4. 更新 `pdf/` 目录下的简历PDF文件
